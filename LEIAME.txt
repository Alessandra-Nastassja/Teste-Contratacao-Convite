AVALIAÇÃO
=========

Seu objetivo neste exercício é criar o CSS de um dos convites da Festalab.

# Regras:
1 - O único arquivo que pode ser modificado é "app.css";
2 - Não é permitido o uso de nenhuma biblioteca extra, apenas recursos CSS suportados pela versão mais recente do Chrome;
3 - A imagem "preview.png" deve ser usada como referência para o resultado esperado.

Quando você finalizar envie o arquivo "app.css" para gabriela@festalab.com, com cópia para breno@festalab.com.
